Initial run of the ToolFactory docker container requires extra steps!
After building or starting this container for the very first time, please wait until everything is fully installed - it can take 10-15 minutes or more.
When the output from docker logs -f [container id] settles down, please restart Galaxy using the supplied restart script:
docker exec [container id] sh /usr/local/bin/restartall.sh

There should be a new history containing the demonstration tools as jobs you can rerun and adjust
There is also a workflow available that will re-create that history and in the process, install those demonstration tools into the Galaxy instance you are using in the ToolFactory section. The supplied samples can be copied and reused or you can upload your own fasta, text and fastqsanger and use those for tool test generation

Each example job demonstrates a variation of the tool patterns that can be generated. The sample tool generating jobs in the history can be re-run and settings adjusted to experiment with the ToolFactory settings for each tool
  • Hello runs a bash script with a single user supplied parameter - "World" for example. STDOUT goes to the history
  • Tacrev uses STDOUT and STDIN without parameters.
  • Two simple python scripts to reverse lines in a text file are driven with (pyrevargparse) argparse style and with (pyrevpos) positional parameters
  • The perl GC counter takes positional parameters and is taken straight from the Planemo documentation tool examples
  • The planemo advanced bwa example is shown in two versions. One uses the original Planemo tutorial code to demonstrate a command over-ride. An equivalent tool but using bash and positional parameters might be easier to maintain since it doesn't use cheetah markup.
  • The Sed example shows how unix utilities like sed - or appropriate Conda package - can be driven using positional parameters
  • The Evil_script_that_fails example tries to create a file where it should not. It should build ok but an error should be noted in the history tgz after it runs.
