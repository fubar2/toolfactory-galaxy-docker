Hello, your Galaxy ToolFactory Docker container is running!

Important: first run steps

Before using this container, please wait until everything is installed - it can take 10 minutes or more. When the output from

docker logs -f [container id]


settles down, please restart Galaxy - find the container id (with "docker ps") and try:

docker exec [container id] supervisorctl restart galaxy:

There is a workflow you need to run and some datasets that should be in a nice new history when you login as admin@galaxy.org password is password
It should create some demonstration tools.

After the workflow has run


There will be 8 new tools installed with the ToolFactory.
Each demonstrates a variation of the patterns that can be generated using the ToolFactory.
  • Hello runs a bash script with a single parameter - "World" for example
  • Tacrev uses STDOUT and STDIN without parameters.
  • Two simple python scripts to reverse lines in a text file are driven with (pyrevargparse) argparse style and with (pyrevpos) positional parameters
  • The planemo advanced bwa example is shown in two versions. One uses the original Planemo tutorial code to demonstrate a command over-ride. An equivalent one using bash and positional parameters might be easier for some programmers.
  • The Sed example shows how utilities like sed - or any Conda package - can be driven using positional parameters
