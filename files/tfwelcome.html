Hello, your Galaxy ToolFactory Docker container is running!

There is a workflow you need to run and some datasets that should be in a nice new history when you login as admin@galaxy.org
It should create some demonstration tools.
Each demonstrates a way of using the ToolFactory. Hello runs a bash script with a single parameter. Tacrev uses STDOUT and STDIN without parameters.
Two simple python scripts to reverse lines in a text file are driven with (pyrevargparse) argparse style and with (pyrevpos) positional parameters.
The planemo advanced bwa example is shown in two versions. One uses the original Planemo tutorial code to demonstrate a command over-ride. An equivalent one using bash and positional parameters might be easier for some programmers.
The Sed example shows how utilities like sed - or any Conda package - can be driven using positional parameters
